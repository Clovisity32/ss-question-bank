Question Bank Admin

Upload & classify papers on this device, edit anything, then publish. You can also manage (edit / delete) questions already on the live site.

← Back to bank

1. Publish settings

Publishing/editing writes to your GitHub repo, so only you (with the token) can change the bank. The token stays in this browser. Create a fine-grained token with Contents: Read & write on this repo at github.com/settings/personal-access-tokens.

2. Upload paper(s)

Pick one or more .docx papers (parsed on-device). Review and edit every field, remove any you don't want, then publish.

3. Manage published questions

Load what's currently live, fix any field, or delete entries (e.g. a wrongly-parsed paper), then save.
